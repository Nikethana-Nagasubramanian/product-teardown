# Product Teardowns

I try out different products and build minimal interactive prototypes for the UX fixes I come up with. Each teardown lives as its own route — click through them to see the redesigns in action.

**Portfolio:** [www.itsmenike.com](https://www.itsmenike.com)

## Teardowns

- **GPTZero** — Redesigning the Advanced Scan experience with bi-directional text ↔ analysis navigation
- **Hume Interaction** — Rethinking call monitoring to surface actionable insights instead of raw emotional data

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Built With

- Next.js · React · TypeScript · Tailwind CSS
